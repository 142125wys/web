云南十大著名旅游景点

导语
INTRODUCTION
云南有着非常多的旅游景点，是一个让人魂牵梦萦的地方。这里不仅有钟灵毓秀的云南石林风景名胜区、九乡风景名胜区，还有充满浪漫色彩的大理·苍山·洱海、玉龙雪山等等，此外，云南由于其独特的地理风貌，特殊的气候状况，多彩的民族风情，奇特的风俗习惯，产生了许多不同于其他地方的奇异现象。

01
云南石林风景名胜区
31112 0
世界地质公园国家风景名胜区国家地质公园国家生态旅游示范区世界遗产
云南省昆明市石林彝族自治县石林中路延长线
该图片由注册用户"学哥"提供，版权声明反馈
云南石林是一个以岩溶地貌为主体的风景名胜区，在石林广达400平方公里的区域内，遍布着上百个黑色大森林一般的巨石群。云南石林拥有世界上喀斯特地貌演化历史较久远、分布面积较广、类型齐全、形态独特的古生代岩溶地貌群落石林，被誉为 “天下第一奇观”。石林喀斯特地貌面积达1100多平方公里，主要由石林风景区、乃古石林风景区、大叠水风景区、长湖风景区、圭山国家森林公园等组成。<详细>
02
九乡风景名胜区
13530 0
国家风景名胜区4A景区
云南省昆明市宜良县九乡彝族回族乡
九乡风景区是以溶洞景观为主体，洞外自然风光、人文景观、民族风情融为一体的综合性风景名胜区。景区内森林茂密，动植物资源丰富，既有壮丽的山川峡谷，也有瑰丽的地下溶洞，属于典型的喀斯特地貌景观。景区拥有上百座大、小溶洞，为国内规模最大、数量最多、溶洞景观最奇特的洞穴群落体系，是独具特色的观光型喀斯特地质公园，被称为溶洞之乡。<详细>
03
大理·苍山·洱海
7841 0
国家风景名胜区
云南省大理白族自治州大理市古城
大理风景名胜区曾是古代南诏国和大理国的都城，总面积1016平方公里，集秀丽的自然风光、悠久的历史文化、浓郁的民族风情、宜人的气候为一体，是进行民俗考察、学术活动的的旅游观光和度假胜地。景区素以苍山洱海为中心的湖泊山岳著称，苍山十九峰，巍峨雄壮，与秀丽的洱海风光形成强烈对照，给人以宁静而悠远的感受。其他著名的景点还有崇圣寺三塔、大理古城、洋人街等。<详细>
04
普达措国家公园
15215 0
5A景区国家公园
云南省迪庆藏族自治州香格里拉市建塘镇红坡村
普达措国家公园由国际重要湿地碧塔海自然保护区和“三江并流”哈巴片区之属都湖景区两部分构成，以碧塔海、属都湖和弥里塘亚高山牧场为主要组成部分，也是香格里拉旅游的主要景点之一。普达措国家公园拥有地质地貌、湖泊湿地、森林草甸、河谷溪流、珍稀动植物等，原始生态环境保存完好。<详细>
05
三江并流-老君山
3371 0
世界遗产
云南省丽江市玉龙纳西族自治县
三江并流风景名胜区是中国境内面积最大的世界遗产地，拥有怒江、澜沧江、金沙江3个风景片区，8个中心景区，60多个风景点，总面积3500多平方公里。三江并流还是世界上罕见的高山地貌及其演化的代表地区，也是世界上生物物种最丰富的地区之一。在众多景点中maigoo小编重点推荐丽江老君山，其拥有丰富的高山植被，珍稀动植物，众多的冰蚀湖，奇异的丹霞地貌和多姿多彩的民风民俗，极具观赏价值和科学考察价值。<详细>
06
玉龙雪山·蓝月谷瀑布·冰川公园
27898 0
5A景区国家风景名胜区国家地质公园
云南省丽江市玉龙纳西族自治县
该图片由注册用户"学哥"提供，版权声明反馈
玉龙雪山景区面积为415平方公里，由十三座雪峰组成，由北向南呈纵向排列，延绵近50公里，自然资源丰富，我们认为最具观赏价值的属蓝月谷和冰川公园，蓝月谷以其巍峨壮美的雪山、湛蓝如玉的湖水与雄齐秀丽的峡谷风光享誉世界；冰川公园全长2.9公里，其末端是千姿百态的冰塔林，因阳光折射呈现出蓝绿色，人称“绿雪奇峰”，这些独具特色的自然景观也是丽江的网红打卡地。<详细>
07
迪庆梅里雪山景区
23523 0
4A景区世界遗产
云南省迪庆藏族自治州德钦县
梅里雪山，是位于西藏察隅县东部与云南迪庆藏族自治州德钦县境云岭乡西部的一座南北走向的庞大的雪山群，全长有150公里。梅里雪山也是雍仲苯教圣地，和西藏的冈仁波齐、青海的阿尼玛卿山、青海的尕朵觉沃并称为藏传佛教四大神山。梅里雪山的“日照金山”景观更是让人称绝，连绵不绝的雪山在日出的映衬下变得金碧辉煌，被《中国国家地理》杂志评为“中国最美的十大名山”。<详细>
08
罗平九龙瀑布群
8077 0
4A景区
云南省曲靖市罗平县长底乡
九龙瀑布群，由于特殊的地质构造和水流的千年侵蚀，在此形成了十级高低宽窄不等，形态各异的瀑布群，或雄伟、或险峻、或秀美、或舒缓，无与伦比美不胜收，瀑间浅滩深潭千姿百态、异彩纷呈，其中最大的一级宽112米，高56米，气势恢宏，蔚为壮观，号称九龙第一瀑。景区内一年四季美景不断，素有“九龙十瀑，南国一绝”的美誉。<详细>
09
阿庐·古洞
3789 0
国家风景名胜区国家地质公园
云南省红河哈尼族彝族自治州泸西县城北部
阿庐风景名胜区荟萃了阿庐古洞、吾者温泉、城子古村、阿拉湖、九溪山森林公园等五十余个景点，以喀斯特地下岩溶和江、湖、瀑、泉自然景观为主体，辅以原汁原味民族文化，形成了具有生态景观与人文景观和谐优美的人居环境和富有特色的生态文化旅游区，是一个休闲、度假、康体的好去处。核心区阿庐古洞被誉为“云南第一洞”，洞内四大自然奇观--佛光惊现、天造神物、古洞云海、地河幻景，十分罕见。<详细>
10
丽江泸沽湖景区
20516 0
4A景区国家水利风景区
云南省丽江市宁蒗彝族自治县永宁乡
泸沽湖旅游景区四周崇山峻岭，，森林资源丰富，山清水秀，空气清新，景色迷人，被当地摩梭人奉为“母亲湖”。湖泊略呈北西一东南走向，南北长9.5公里，东西宽5.2公里，湖岸线长约44公里。它犹如一块明珠镶嵌在群山怀抱之中，碧波荡漾，风光迷人，有“高原明珠”、“滇西北的一片净土”、“东方第一奇景”等美称。这里古朴的民风，秀丽的山光水色与浓郁的传奇风情，充满了神秘的色彩。<详细>
